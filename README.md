# Lang Project

A C++ project built with CMake.

## Building the Project

1. Create a build directory:
```bash
mkdir build
cd build
```

2. Generate the build files:
```bash
cmake ..
```

3. Build the project:
```bash
make
```

4. Run the executable:
```bash
./LangProject
```

## Project Structure

```
lang/
├── CMakeLists.txt
├── src/
│   └── main.cpp
└── README.md
```

## Requirements

- CMake 3.10 or higher
- C++17 compatible compiler (GCC, Clang, etc.)
